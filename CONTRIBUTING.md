# Contributing

Thanks for your interest in improving this list of image to video tools.

This repository aims to be:

- Simple and easy to read
- Focused on image to video (with some text to video overlap)
- Helpful for creators, marketers, and developers

Please follow the guidelines below when adding or updating tools.

---

## What you can contribute

You are welcome to:

- Add a new image to video tool
- Update a tool description, link, or free tier information
- Remove tools that are dead, paywalled, or no longer relevant
- Fix typos or improve clarity

---

## Before you add a tool

A tool is a good fit for this list if:

1. It has a public website or app that anyone can access  
2. It supports image to video in some form  
3. It has a free tier, free trial, or free credits that new users can actually use  
4. It is not pure spam, not a referral/affiliate link farm  

Nice to have but not required:

- Clear pricing page
- Examples or demo gallery
- API or SDK

---

## How to add a new tool

1. Fork this repository  
2. Edit `README.md`  
3. Find the `Tool list – Free or Freemium Image to Video Platforms` table  
4. Add a new row at the end of the table with this format:

   - Keep the numbering sequential  
   - Keep the name short  
   - Use the official URL if possible  
   - Write a neutral, factual description  
   - Mention how the free tier works in a single short phrase
