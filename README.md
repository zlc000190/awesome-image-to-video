# Awesome Free Image to Video Tools (2025)

A curated list of free and freemium AI image-to-video tools that turn still images into short videos.

This list focuses on:

- Online tools and web apps that support image to video
- AI video generators that also do text to video or video to video
- Practical use cases such as social media clips, UGC ads, and ecommerce product promos

If you are looking for a place to compare different image-to-video platforms before spending money on credits and subscriptions, this repository gives you a fast overview.

## Evaluation criteria

A tool is included if:

- It supports image-to-video in a browser or via a simple UI  
- It has a free tier, free trial, or free credits that a new user can realistically use  
- The service is open to the public and not only private beta  

Nice-to-have points:

- Clear pricing page  
- Public examples or demo gallery  
- API or SDK for developers
